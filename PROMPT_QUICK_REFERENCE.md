# 提示词快速参考卡 (Quick Reference)

## 🎯 5步翻译决策树

```
STEP 1: 字面翻译检查
  ↓ 能自然说出? → YES → 使用字面
  ↓ 不自然? → GO TO STEP 2

STEP 2: 习语检测
  ↓ 有动物/身体部位/天气? → 肯定是习语
  ↓ 短语无逻辑意思? → 可能是习语
  ↓ 无信号? → 用字面

STEP 3: 习语翻译
  ↓ 说话者真正的意思是什么?
  ↓ 找中文等价表达

STEP 4: 自然性检查
  ↓ 本地人会这样说? → YES → 继续
  ↓ 不自然? → 调整

STEP 5: 一致性检查
  ↓ 与前文术语匹配? → YES → 输出
  ↓ 不匹配? → 更新
```

## 💡 常见问题速答

**Q: 什么时候是习语?**  
A: 有动物/身体部位 = 100%习语。短语无逻辑意思 = 很可能。

**Q: 上下文什么时候有用?**  
A: 同角色(声线) + 同话题(术语) + 同场景(情感)

**Q: 为什么这个字幕长?**  
A: 因为中文需要更多字来表达英文的习语或文化含义

**Q: 如何处理人名?**  
A: 保留原始，或用建立的翻译(一致性第一)

**Q: 什么时候用粗鲁表达?**  
A: 角色性格 + 上下文(绝不为了粗鲁而粗鲁)

## 📊 性能优化建议

### 如果翻译太慢:
- [ ] 降低上下文条数 (context -= 1)
- [ ] 简化体裁指导 (genre = general)
- [ ] 减少提示词长度

### 如果质量下降:
- [ ] 提高上下文条数 (context += 1)  
- [ ] 增加体裁指导细节
- [ ] 检查temperature是否太高

### 如果习语处理不好:
- [ ] 增加源语言的习语例子
- [ ] 确保检测信号清晰
- [ ] 增加质量检查严格度

## 🚀 高级用法

### 添加自定义术语库

```angelscript
// 在GetGenreSpecificGuide()中添加
string customTerms = "CUSTOM TERMS FOR " + genre + ":\n";
customTerms += "术语1 → 标准翻译\n";
customTerms += "术语2 → 标准翻译\n";
prompt += customTerms;
```

### 针对特定用户群优化

```angelscript
// 示例: 游戏翻译人员
IF genre == "gamedev"
  prompt += "AUDIENCE: Game developers\n"
          + "Technical accuracy CRITICAL\n"
          + "Use official engine terminology\n";
```

### 实时调整温度

```angelscript
// 基于内容类型
IF genre == "anime"
  temperature = 0.4;  // 更多创意
ELSE IF genre == "gamedev"
  temperature = 0.2;  // 更保守
```
